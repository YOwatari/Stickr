Stickr
======

ステッカー試し張りサービス BananaChoco@DMTC☓MS

SETUP
======

要：node.js, MongoDB

```
npm install -g yo
git clone git@github.com:YOwatari/Stickr
cd Stickr
npm install && bower install
```

MongoDB起動
```
cd Stickr
mkdir mongo
mongod --dbpath mongo
```

Webアプリ起動
```
grunt serve
```

Open http://localhost:5000
